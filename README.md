Saya sedang dalam misi mengeksplorasi dunia pengembangan web dengan bantuan AI. Fokus saya adalah memahami bagaimana teknologi masa kini bisa mempercepat proses belajar dan pembangunan aplikasi.

### 🚀 Apa yang sedang saya kerjakan?

- ⚙️ **Backend:** Memahami logika server-side tanpa pusing di awal.
- 🤖 **AI Integration:** Bereksperimen dengan AI untuk menulis kode yang lebih efisien.
